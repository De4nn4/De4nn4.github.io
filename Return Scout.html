Return Scout

Attention Restoration Theory proposes that natural environments can restore mental fatigue from directed attention. Return Scout is a proposed Mock-up video for a wellness app, backed by this theory. I designed and edited together every asset, to help showcase how the application would have worked if it were fully developed. The result shows how Return Scout plans to reconnect users to a childhood experience of nature through the tools the application offers.
